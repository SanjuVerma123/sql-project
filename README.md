# 🛒 E-Commerce SQL Data Analysis Project

## 📌 Overview

This project focuses on cleaning, transforming, and analyzing an e-commerce dataset using SQL.

## ⚙️ Tools Used

* SQL (MySQL)
* GitHub

## 📂 Project Structure

* `schema/` → Database and table creation
* `data_cleaning/` → Data cleaning & preprocessing
* `analytics/` → Business insights & analytics
* `data/` → Raw dataset
* 'Ecommerce_Sales_Insights_SQL_Project' - Complete Project

## 🔍 Key Tasks Performed

* Data cleaning (NULL handling, trimming, formatting)
* Date standardization
* Data deduplication using window functions
* Customer segmentation
* Revenue trend analysis

## 📊 Insights Generated

* Total revenue & profit
* Monthly sales trends
* Customer segmentation (VIP, Regular, Low Value)
* Top-performing products

## 🚀 How to Run

1. Run `schema.sql`
2. Import dataset into `ecommerce_table`
3. Run `data_cleaning.sql`
4. Run `analytics.sql`

## 📈 Skills Demonstrated

* SQL data cleaning
* Window functions
* Business analytics
* Data transformation
